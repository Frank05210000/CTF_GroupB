# 你以為你登進去了

## Challenge

這是一個 TW-Corp 內部員工系統。你有一組普通員工帳號，但聽說系統裡有管理者才能看到的機密資料，裡面藏著 flag。

請登入系統並取得 flag。

## Target

```text
http://<Linux主機IP>:5000
```

## Account

```text
employee / pass1234
```

## Hint

登入只是開始。URL 上的資料識別碼可能比頁面本身更重要。

## Rules

- 只能攻擊題目提供的 Web 服務。
- 不需要暴力破解。
- 不需要 DoS。
- 不需要攻擊其他主機或服務。
