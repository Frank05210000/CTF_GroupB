CTF GroupB 題目入口

請閱讀每題敘述，從指定目標開始測試並取得 flag。所有攻擊範圍都限制在本 CTF lab 服務內。

Flag 格式為 FLAG{...}。若部署在遠端 Linux lab 主機，請將頁面中的 127.0.0.1 換成該主機 IP。

200 pts Web IDOR

這是一個 TW-Corp 內部員工系統。你有一組普通員工帳號，但聽說系統裡有管理者才能看到的機密資料。

起點
http://127.0.0.1:5000
帳號
employee / pass1234
任務

提示
登入只是開始。URL 上的資料識別碼可能比頁面本身更重要。

300 pts SSH Privilege Escalation

你拿到一台隔離 lab 主機的目標服務。請在指定範圍內進行偵察，取得系統內的 root flag。

起點
127.0.0.1:2222
任務
先確認服務類型，再取得低權限 shell，最後嘗試讀取 root flag。
提示
The first door is easy to knock. The real flag is deeper.

共同規則

  • 只能攻擊本頁列出的 lab 服務。
  • 禁止 DoS 或攻擊 lab 外任何主機。
  • 不需要修改 VM 桌面或讀取發題資料夾。
  • 不需要登入出題者帳號；所有題目資訊都在本頁提供。
